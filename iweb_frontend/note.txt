一、浮动布局坑
假设:.parent>.child+.child2+.child3
.parent宽度可以容纳.child1和.child3，如果三个子元素都浮动的话，.child3会被挤到下一行
情形0:.child1和.child2一样高，则.child3跑到.child1下方
情形1:.child1比.child2高，则.child3跑到.child2下方
情形2:.child1和.child2一样矮，则.child3跑到.child1下方，
但顶部与.child2下方找齐（即：.child3不会挨到.child1)

二、仿Bootstrap布局系统
  gutter：槽宽，指两列内容之间的间距，一般由相邻的两列各贡献一半的padding组合而成
  .col-*-* {
	  float: left;
	  padding: 0  1em;     /*此处的1em就是一半的槽宽*/
  }
  注意：为了清楚整行最开始的列最左侧的留白，以及整行结束的列最右侧的留白，可以将整行进行margin的调整
  .row {
	  margin-left: 1em;
	  margin-right: 1em;
  }
